## How to use

**HeyDoc!** is simple to use. Just create `.md` or `.html` pages in the `/docs` repository.


HeyDoc structure
```
├── bin
│   └── heydoc
├── docs
│   ├── examples
│   │   ├── custom.md
│   │   ├── github_flavored_markdown.md
│   │   ├── html_file.html
│   │   └── simple.md
│   ├── getting_started.md
│   ├── index.md
│   └── settings.yml
└── web
    ├── .htaccess
    └── index.php
```
