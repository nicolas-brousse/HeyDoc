# Themes

This is a list of themes available with HeyDoc.



## How choose

The default theme is `default` but you can choose an other theme. Themes into this folder are the only available.



## For developpers


### Accessible vars

* `app`
  * `app.config`
  * `app.request`
* `page`
* `content`



### Twig helpers

#### Methods

* `heydoc_version()`

#### Filters

* `|markdown_transform`
