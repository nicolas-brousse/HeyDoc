# HeyDoc


## Installation

Crete `composer.json` file:

```json
{
    "name": "heydoc/heydoc-application",
    "license": "MIT",
    "type": "project",
    "description": "HeyDoc application",
    "require": {
        "php": ">=5.3.3",
        "heydoc/heydoc": "@dev"
    },
    "config": {
        "bin-dir": "bin"
    }
}
```

And `composer install`.
